## RFM 비교 표 (주요 기업)

RFM 지표는 **회사별로 서로 다른 접근**을 사용하고 있기 때문에, 단일 척도로 비교하기보다는 **RFM 요소별(데이터·모델·추론·오픈소스 등)**로 비교하는 것이 적합합니다.

| 기업 | RFM 접근 방식 | 핵심 특징 | 비고 |
|---|---|---|---|
| Tesla (Optimus Gen3) | 자체 RFM (FSD 기반) | VLA 전용 HW/SW 통합, 제조 데이터 기반 | 에지 추론/학습 데이터 중심 |
| Figure AI (Figure 03) | 자체 RFM (Helix VLA) | 카메라·센서·핸드 동시 설계, 고밀도 모델 | 정밀 조작·기능 확장 지향 |
| Boston Dynamics (Atlas) | 외부 RFM + 자체 통합 (Gemini) | 56DOF 물리 최적화, 로봇 특화 모델 | 협업/현장 테스트 중심 |
| Google DeepMind | 외부 RFM (Gemini Robotics) | 대규모 VLA + 추론 최적화 | 설명 가능성·데이터 중심 |
| Unitree | 자체 RFM (UnifloM-VLA) | 경량 플랫폼, 오픈 SDK 생태계 | 빠른 시연·저가 시장 지향 |
| LG (KAPEX/CLOiD) | 자체 RFM (KAPEX) | 가정/산업용 통합 플랫폼 | 생태계·파트너 연계 시도 |

> ⚠️ 현재 데이터 구조는 주로 **RFM 스코어(R/F/M 6개 요소)**를 기준으로 계산됩니다. 평가는 위 표처럼 각 회사가 어떤 **데이터/모델/컴퓨팅/오픈소스 전략**을 쓰는지에 따라 달라질 수 있으므로, **회사별 RFM 요소별 비교**를 우선시하는 편이 타당합니다.
